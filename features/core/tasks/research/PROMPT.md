# Research Prompt

Research a feature to gather relevant information from the internet.

## Context Provided

- **GOAL.md**: Feature goals
- **Source Files**: Current implementation (if any)

## Your Task

Research based on GOAL.md:

1. **Best Practices**: Industry standards for this type of feature
2. **Libraries & Tools**: Relevant packages or frameworks
3. **Similar Implementations**: Patterns from similar projects
4. **Potential Challenges**: Common pitfalls

## Output Guidelines

- Be concise - no redundant information
- Use bullet points, not paragraphs
- Only include actionable findings
- Skip sections with no relevant findings

## Output Format

```markdown
# Research: [Feature Name]

## Summary
One-line overview.

## Best Practices
- ...

## Libraries/Tools
- ...

## Challenges
- ...

## Sources
- [Title](url)
```
